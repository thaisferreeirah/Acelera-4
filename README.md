# Entr.ai

Este é um projeto em grupo desenvolvido no 4º semestre do curso de Análise e Desenvolvimento de Sistemas.

O desafio proposto para este projeto foi desenvolver um sistema utilizando o Arduino integrado com uma API para comunicação entre a página web, banco de dados e o dispositivo hardware.

### Descrição
O projeto propõe um sistema de controle de entrada em uma instituição através do reconhecimento facial. É realizado o cadastro da foto de uma pessoa e através da captura da câmera, é feito o reconhecimento facial, liberando o acesso à instituição. Através da página web é possível monitorar o histórico de acesso, ativar ou desativar o reconhecimento facial, liberar o acesso manualmente e realizar o cadastro de autorizados.

### Tecnologias
- HTML, CSS e JavaScript
- Python
- PostgreSQL

### Materiais
- ESP32 WROOM
- OV2640 + ESP32 MB (Câmera)
- SG90 (Motor do portão)
